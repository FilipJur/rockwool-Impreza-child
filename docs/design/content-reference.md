# Content Reference

## Text Content with Style Mappings

### Hero Section
```
"odměny se hrnou, když zatepluješ vlnou"          → text-hero (UPPERCASE)
"Registrujte se a získejte první z mnoha odměn"   → text-button-lg
"Věrnostní program Mistr Fachman je určen..."     → text-caption
```

### Main Content
```
"Zateplujete izolací ROCKWOOL?"                    → text-h1
"V tom případě se přednostně zaregistrujte..."    → text-body-bold
"Věrnostní program Mistr Fachman odstartuje..."   → text-body
```

### Benefits Cards


### Expert Tip
```

```

### CTA Section
```
"Vyplňte své údaje, zaregistrujte se..."          → text-h2 (white)

"Chcete vědět více? Podívejte se na podmínky..."  → text-fine (white)
```
